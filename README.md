# Portfolio — Sound Engineer (Next.js + Tailwind)

Style: **D2 — Gradient Soft (futuriste, clean)**

## Quick start
```bash
npm install
npm run dev
# open http://localhost:3000
```

## Replace content
- Showreel: edit `src/app/reel/page.tsx` (ReactPlayer `url`)
- Projects: edit `content/projects.json` and images in `public/images/projects`
- Demos: edit `content/demos.json` and put MP3s in `public/audio`
- About photo: replace `public/images/me.jpg`
- Contact email: edit `src/app/contact/page.tsx`

## Deploy (free)
1. Push to GitHub
2. Import the repo on **Vercel**
3. Deploy — get your `*.vercel.app` URL

Enjoy!
